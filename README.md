# Site portfolio Liqoy

Site one-page moderne en HTML/CSS/JS inspiré d'un design sombre pour présenter **Liqoy**.

## Fichiers

- `index.html` : structure de la page.
- `styles.css` : thème sombre moderne (cartes, tags, responsive).
- `script.js` : scroll fluide vers les sections et année automatique dans le footer.

## Utilisation

Ouvre simplement `index.html` dans ton navigateur.

Pour changer tes liens (GitHub, Discord, YouTube, X/Twitter), édite la section **Portfolio** dans `index.html` (les balises `a` dans le bloc `links-grid`).
